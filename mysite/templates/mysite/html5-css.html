Seletores CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML ou XML.

Seletor Usado Comentário
Seletor de Elemento check_circle Seleciona elementos pelo nome da tag. Utilizado para estilizar todos os parágrafos nesta página.
Seletor de Classe check_circle Seleciona elementos que possuem um atributo class específico. Utilizado para estilizar botões nesta página.
Seletor de ID check_circle Seleciona elementos pelo seu atributo id. Utilizado para estilizar um elemento específico nesta página.
Seletor de Atributo check_circle Seleciona elementos com um atributo específico. Utilizado para estilizar links externos nesta página.
Seletor Universal check_circle Seleciona todos os elementos. Utilizado para aplicar margens padrão nesta página.
Seletor de Pseudo-classe check_circle Seleciona elementos em um estado específico. Utilizado para estilizar links visitados nesta página.
Seletor de Pseudo-elemento check_circle Seleciona e estiliza uma parte específica de um elemento. Utilizado para estilizar a primeira letra de parágrafos nesta página.

Combinadores de seletores

Combinador Usado Comentário
Descendente (espaço) check_circle Seleciona elementos que são descendentes de outro elemento. Utilizado para estilizar parágrafos dentro de divs nesta página.
Filho (>) check_circle Seleciona elementos que são filhos diretos de outro elemento. Utilizado para estilizar listas dentro de uma seção nesta página.
Adjacente (+) check_circle Seleciona elementos que são adjacentes a um determinado elemento. Utilizado para estilizar elementos após um header nesta página.
General Sibling (~) check_circle Seleciona elementos que são irmãos de um determinado elemento. Utilizado para estilizar todos os parágrafos após uma div específica nesta página.
